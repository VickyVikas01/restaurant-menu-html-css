# Restaurant Menu Layout (HTML + CSS)

This is a simple restaurant menu webpage created using:
- HTML5
- CSS3 (Flexbox, Tables)

Features:
- 6 menu categories (Veg, Breads, Non-Veg, Rice, Chinese, South Indian)
- Image + table layout for each category
- Responsive flexbox grid layout
- Styled with black & yellow theme

Purpose:
Practice HTML tables, semantic tags and CSS flexbox layout.
